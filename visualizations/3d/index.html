Particles

A particle visualization produces a three dimensional scatter plot using three.js. The camera can be controled by using the WASD and arrow keys. Use WASD to move the camera's position, and use the arrow keys to change the direction of the focus (FPS style controls).

Volume

Render a collection of images as a three dimensional volume. The images are overlayed on eachother transparently to create a three dimensional effect. This uses the same FPS-style controls as the particles visualization.

  • Python Example
from PIL import Image
import requests
from StringIO import StringIO

image_urls = ["http://i.gif.fm/janelia-images/stack-2/plane%s.png" % i for i in range(1, 42)]

images = []
for image_url in image_urls:
    response = requests.get(image_url)
    img = Image.open(StringIO(response.content))
    images.append(img)
    
lgn.volume(images)
