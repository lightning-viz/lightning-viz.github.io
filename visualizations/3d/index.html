Scatter3

A scatter3 visualization produces a three dimensional scatter plot using three.js. The camera can be controlled by using the WASD and arrow keys. Use WASD to move the camera's position, and use the arrow keys to change the direction of the focus (FPS style controls).

  • Data Structure
  • Python Example
data = {
    "points": [[x0,y0,z0], [x1,y1,z0], ... ]
    "color": [[r0,g0,b0], [r1,g1,b1], ...]
    "label": [l0, l1, ...]
}

from numpy import random, column_stack

n = 500
x = random.rand(n)*255
y = random.rand(n)*255
z = random.rand(n)*255
c = column_stack((x, y, z))

lgn.scatter3(x, y, z, color=c, size=3)

Volume

Render a collection of images as a three dimensional volume. The images are overlayed on eachother transparently to create a three dimensional effect. This uses the same FPS-style controls as the particles visualization.

  • Python Example
from numpy import ndarray, linspace

nx, ny, nz = (50, 50, 50)
rgb = ndarray((nx,ny,nz,3))
hsv = ndarray((nx,ny,nz,3))
for i, ii in enumerate(linspace(0,1,nx)):
    for j, jj in enumerate(linspace(0,1,ny)):
        for k, kk in enumerate(linspace(0,1,nz)):
            position = (i, j, k)
            rgb[position] = (kk, jj, ii)
            
lgn.volume([x for x in rgb])
